# 이듬 회귀 QA 체크리스트

패치할 때마다 아래 동선을 확인합니다.

## 핵심 동선

### 한줄 피드
- [ ] 피드 로드 + 문장이 첫 시선
- [ ] 스크롤 후 상세 진입 → 뒤로 → 스크롤 위치 유지
- [ ] 피드 → 책 페이지 → 뒤로 → 스크롤 위치 유지
- [ ] 전체/구독 필터 전환
- [ ] 더보기 메뉴 → 안 보기 → 숨겨진 게시글 + 되돌리기
- [ ] 책 제목 클릭 → 책 페이지 이동

### 기록
- [ ] 새 기록: 제목 → 문장 → 감상 → 출처 → 기록하기
- [ ] 기록 후 홈 피드에 즉시 반영
- [ ] 임시 저장: 출처 없이 문장만 → "임시로 남기기"
- [ ] 임시 저장 후 서재 "남겨둔 문장"에 표시
- [ ] 임시 문장 클릭 → 수정 → 출처 추가 → 기록하기 → 홈 반영
- [ ] 임시 문장 삭제 (확인창)
- [ ] 상세에서 수정 → 기존 내용 채워짐 → 수정하기 → 복귀

### 노트
- [ ] 새 노트 만들기 (서재 또는 노트 탭)
- [ ] 노트 에디터: 기록 추가, 내 글 쓰기, 구분선
- [ ] 에디터 헤더 고정 (스크롤해도 보임)
- [ ] 에디터 등록 → 공개 + 이동
- [ ] 에디터 ··· → 미리보기/공개·비공개/삭제
- [ ] 감상(feeling) 편집 → DB 반영 → 리더에서 표시
- [ ] 자동 페이지 구분선 표시
- [ ] 드래그 정렬 (hover 시 핸들 표시)
- [ ] 노트 리더: 표지 → 내용 → 엔딩
- [ ] 리더 상단 탭 → 헤더 토글 (이북 방식)
- [ ] 리더 좌우 넘김 + 스와이프 + 키보드
- [ ] 리더 ··· → 링크 복사 / 편집 / 삭제
- [ ] 문장+메모 합침 페이지
- [ ] 짧은 문장 묶음 페이지

### 서재
- [ ] 프로필: 이름, 핸들, 바이오, 등급
- [ ] 대표 문장 / 대표 노트 표시
- [ ] 축적 요약 문장
- [ ] 읽는 책 / 노트 / 남긴 문장 섹션
- [ ] 남겨둔 문장 (임시) 표시 + 삭제
- [ ] 등급 클릭 → 등급 안내 바텀시트
- [ ] 프로필 편집 링크
- [ ] + 새 노트 버튼

### 검색
- [ ] /search 진입 (사이드바 링크 또는 헤더)
- [ ] 최근 검색어 표시 + 클릭
- [ ] Enter → 결과 화면 (전체/문장/책/사람/노트 탭)
- [ ] 문장 결과 → 한줄 상세로 이동
- [ ] 사람 결과 → 프로필로 이동

### 공유카드
- [ ] 상세에서 카드 만들기
- [ ] 스타일/폰트/비율 선택
- [ ] 저장 + 공유

### 프로필 (타인)
- [ ] 구독 → "구독을 시작했습니다" 토스트
- [ ] 노트 3개 이상 → 전체 보기
- [ ] 한줄 목록 표시

## 크로스 체크

- [ ] @@ 이중 핸들 없음 (URL에 @ 하나만)
- [ ] 비로그인 시 기능 접근 → 로그인 모달
- [ ] Escape 키로 모달 닫기
- [ ] 토스트 모바일 잘림 없음
- [ ] 에디터 헤더 사이드바와 겹치지 않음
- [ ] 노트 리더 뒤로가기 버튼 보임
