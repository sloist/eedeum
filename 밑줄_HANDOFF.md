# 밑줄 (Underline) — 프로젝트 핸드오프 문서

## 한 줄 정의
**읽은 내가 남는 곳.** 책 속 문장을 찍고, 한 줄 감상을 남기고, 남의 밑줄을 엿보는 독서 기록 SNS.

## 핵심 흐름
찍는다 → 남긴다 → 엿본다

---

## 기술 스택 (예정)
- React + Vite + TypeScript
- Supabase (Auth, DB, Storage)
- Vercel (배포)
- 현재 프로토타입: `밑줄_v4.jsx` (단일 React 컴포넌트, 전체 앱 구조 포함)

## 현재 상태
- **프로토타입 완성 (v4)**: 모든 UI/UX 작동, 샘플 데이터 기반
- **백엔드 없음**: 전부 로컬 state + 하드코딩 데이터
- **단일 파일**: 컴포넌트 분리, 라우팅, 상태관리 미적용

## 페이지 구조 (하단 탭 5개)

### 1. 홈
서브탭 3개:
- **팔로잉**: 팔로우한 사람들의 밑줄 피드 + 상단 트렌딩 책 가로 스크롤
- **발견**: 주제 필터 칩 (성장/사랑/위로/철학/여행/관계/예술/심리) + 필터된 밑줄 피드
- **지금**: 실시간으로 올라오는 밑줄 (LIVE 뱃지 → "방금" 뱃지로 변경됨)

### 2. 탐색
- 검색 입력 (터치 시 검색 오버레이 열림)
- 주제 필터 칩 (홈 발견탭과 동일)
- 책 목록 (밑줄 수 기준 정렬, 주제 필터 적용)
- 책 클릭 → 비교 화면 (같은 책, 다른 밑줄)

### 3. [카메라] (중앙 플로팅 버튼)
- 바텀시트로 올라옴
- Step 1: 사진 촬영 영역 + 직접 입력 필드
- Step 2: AI 인식 결과 프리뷰 + 한 줄 감상 입력 (선택) + "밑줄 남기기" 버튼
- 제출 → 토스트 "밑줄이 서재에 남았습니다"

### 4. 모음 (구 "대화" → 채팅 제거됨)
서브탭 3개:
- **저장한 밑줄**: 북마크한 다른 사람의 밑줄
- **받은 공감**: 내 밑줄에 달린 공감 댓글 목록 (새 공감은 배경색 다름 + 점 표시)
- **좋아요한**: 하트 누른 밑줄 목록
- 하단 탭 아이콘에 새 공감 있으면 작은 점(5px, accent 색) 표시

### 5. 서재 (마이페이지)
- **프로필 헤더 (가로 레이아웃)**:
  - 좌: 아바타 (56px)
  - 우: 이름, 핸들, 바이오, 통계 (책/밑줄/팔로워/팔로잉 가로 한 줄)
- 인스타 공유 카드 (어두운 배경, 문장 + 책 + "밑줄" 로고)
  - 클릭 시 공유 모달 열림
- 월간 히트맵 (깃허브 스타일, 브라운 톤)
- 내 서재 그리드 (4열, 책 커버 컬러 + 밑줄 수)

## 핵심 컴포넌트

### PostCard (밑줄 카드)
```
- 유저 아바타 + 이름 (클릭 → 프로필) + 책 제목·작가 (클릭 → 비교) + 시간
- 인용문 블록 (배경색은 coverColor + 30% opacity)
- 한 줄 감상
- 액션 바: 좋아요(토글) | 공감(토글, 펼침) | 저장(토글) | 공유
- 공감 영역 (접힘/펼침):
  - 기존 공감 목록 (점 + 유저명 + 텍스트)
  - 원탭 칩: "나도 여기에 밑줄" / "이 문장 앞에서 멈춤" / "오래 남을 것 같은" / "꺼내 읽게 되는"
  - 직접 입력 + 전송
- "다른 사람의 밑줄 N개" 토글:
  - 같은 책의 다른 밑줄 카드들
  - "모든 밑줄 보기 →" 링크
```

### ProfileHeader (가로 프로필)
```
- 아바타(좌) + 정보(우) 가로 배치
- 다른 유저 프로필: 팔로우/팔로잉 토글 버튼 추가
```

### ShareModal (공유)
```
- 공유 카드 프리뷰 (어두운 배경)
- 4개 버튼: 인스타 스토리 / 인스타 피드 / 카카오톡 / 링크 복사
```

### SearchOverlay (검색)
```
- 전체화면 오버레이
- 인기 검색어 (검색어 비어있을 때)
- 결과: 책 섹션 + 밑줄 섹션
```

### BookCompare (같은 책, 다른 밑줄)
```
- 책 제목 + 작가
- 페이지 순 정렬된 밑줄 카드 리스트
- 유저명 클릭 → 프로필
```

- id, underline_id, user_id, text, created_at

- user_id, underline_id, created_at
